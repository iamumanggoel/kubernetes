This is a part of BlogApplication(asp.net core web apis .net8, angular 17, postgres db, docker, kubernetes).

This repository containers yaml file "deployment-services.yml) to create delpoyments and services for all the ASP.NET Core Web APIs along with postgres db and frontend(angular). It fetches image of all those projects
from Dockerhub repo "umang2001/blogapplication".

It also has a yml file for ingress controller(load balancer acting as a gateway)
